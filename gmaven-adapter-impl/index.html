GMaven Adapter Implementation

Provides the default gmaven-runtime-api implementation supporting Groovy 2.0+ and 2.1+ runtimes.

Default version: Groovy 2.4.9

New versions of Groovy could break compatiblity, so becareful if customizing the Groovy version. If an incompatiblity is found, please report the problem.
